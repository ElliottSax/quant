#### Task 1: Form LLC (2 hours)

**Options:**

**Option A: DIY (Cheapest - $100-200)**
1. Go to your state's Secretary of State website
2. Search for "LLC formation" or "Business registration"
3. Fill out Articles of Organization
4. Pay filing fee ($50-200 depending on state)
5. Wait for approval (1-5 business days typically)

**States with online filing:**
- Delaware: https://corp.delaware.gov/
- Wyoming: https://sos.wyo.gov/
- Your state: [search "[your state] LLC formation"]

**Option B: Service (Easier - $300-500)**
1. **ZenBusiness** - $0 + state fees + $199/year (recommended)
   - Website: https://www.zenbusiness.com/
   - Includes registered agent (required)
   - Fast processing

2. **Northwest Registered Agent** - $39 + state fees + $125/year
   - Website: https://www.northwestregisteredagent.com/
   - Privacy-focused
   - Excellent service

3. **LegalZoom** - $79 + state fees + extras
   - Website: https://www.legalzoom.com/
   - Well-known but more expensive

**Recommended:** ZenBusiness for balance of cost/service

**Information You'll Need:**
- LLC Name: "[Your Company Name], LLC"
  - Check availability first
  - Should match domain if possible
- Registered Agent: You OR service (recommended service)
- Business Address: Your address OR virtual office
- Members: Just you (single-member LLC)

#### Task 2: Apply for EIN (1 hour)

**What:** Employer Identification Number from IRS (like SSN for businesses)

**Why:** Required for bank account, taxes, contracts

**How:** Online application (FREE, takes 15 minutes)

**Steps:**
1. Go to: https://www.irs.gov/businesses/small-businesses-self-employed/apply-for-an-employer-identification-number-ein-online
2. Click "Apply Online Now"
3. Select "Limited Liability Company (LLC)"
4. Answer questions:
   - LLC name
   - Address
   - Responsible party (you)
   - Number of members (1)
   - Business start date
5. Submit
6. **IMPORTANT:** Download and save EIN letter immediately

**You'll receive EIN instantly** (if during business hours)

**Action Items:**
- [ ] Complete IRS EIN application
- [ ] Download EIN confirmation letter
- [ ] Save EIN in secure location (you'll need this constantly)
- [ ] Print 3 copies of EIN letter

**Cost:** $0 (FREE)

**Time:** 15-30 minutes

#### Task 2: Set Up Cloudflare (2 hours)

**Why:** Free CDN, SSL, DDoS protection, DNS management

**Steps:**

1. **Create Cloudflare Account**
   - Go to: https://dash.cloudflare.com/sign-up
   - Use business email (create one first if needed)
   - Free plan is perfect

2. **Add Your Domain**
   - Click "Add a site"
   - Enter your domain
   - Select Free plan
   - Cloudflare will scan DNS records

3. **Update Nameservers**
   - Cloudflare will provide 2 nameservers
   - Example: `ns1.cloudflare.com` and `ns2.cloudflare.com`
   - Go to Namecheap
   - Domain management ‚Üí Custom DNS
   - Replace nameservers with Cloudflare's
   - Save changes
   - **Note:** Takes 24 hours to propagate

4. **Configure Cloudflare Settings**
   - SSL/TLS: Set to "Full" (will change to "Full Strict" later)
   - Speed ‚Üí Optimization: Enable Auto Minify (JS, CSS, HTML)
   - Caching: Set to "Standard"
   - Security ‚Üí Bot Fight Mode: Enable
