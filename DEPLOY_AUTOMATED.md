# Automated Deployment Guide

## Quick Deploy (2 commands)

```bash
# 1. Authenticate (one-time, opens browser)
./auth.sh

# 2. Deploy everything
./deploy.sh
```

That's it! Your app will be live in ~5 minutes.

---

### Backend (Railway)
- FastAPI application
- Real-time stock quotes (Finnhub)
- Portfolio backtesting engine

- Auto-scaled hosting

**URL**: `https://quant-backend-production.up.railway.app`

### Frontend (Vercel)
- Next.js application
- Portfolio backtesting UI
- Real-time visualizations
- Global CDN delivery

**URL**: `https://quant-frontend.vercel.app`

---

## Authentication Details

### auth.sh does:
1. Opens browser for Railway login
2. Opens browser for Vercel login
3. Saves credentials locally
4. Verifies authentication

**Run once per machine.**

### deploy.sh does:
1. Creates/updates Railway project
2. Sets environment variables (including Finnhub API key)
3. Deploys backend to Railway
4. Deploys frontend to Vercel
5. Connects frontend to backend API
6. Updates CORS settings

**Run every time you want to deploy updates.**

## Continuous Deployment (GitHub Actions)

Already configured! Just push to main:

```bash
git add .
git commit -m "Update app"
git push origin main
```

Auto-deploys on every push to `main` branch.

**Setup Required**:
1. Add secrets to GitHub repo:
   - `RAILWAY_TOKEN` (from Railway dashboard)
   - `VERCEL_TOKEN` (from Vercel settings)
   - `VERCEL_ORG_ID` (from Vercel settings)
   - `VERCEL_PROJECT_ID` (from Vercel settings)

Get tokens:
- Railway: `railway token`
- Vercel: `vercel token` or dashboard → Settings → Tokens

## Troubleshooting

### Issue: "railway: command not found"
```bash
npm install -g @railway/cli
```

### Issue: "vercel: command not found"
```bash
npm install -g vercel
```

### Issue: "Unauthorized" error
```bash
# Re-authenticate
./auth.sh
```

### Issue: Deployment fails
```bash
# Check logs
railway logs  # Backend
vercel logs   # Frontend
```

### Issue: Frontend can't connect to backend
```bash
# Check CORS settings in Railway dashboard
# Should include: https://*.vercel.app
```

### Issue: Finnhub API not working
```bash
# Verify API key in Railway dashboard
cd quant/backend
railway variables
# Should show: FINNHUB_API_KEY=d6fl2j9r01qqnmbp36ogd6fl2j9r01qqnmbp36p0
```
